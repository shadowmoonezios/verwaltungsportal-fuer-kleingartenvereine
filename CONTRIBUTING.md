# Mitwirken

Vielen Dank, dass Sie interessiert sind, zu diesem Projekt beizutragen! Hier sind einige Möglichkeiten: 

## Fehler melden
Wenn Sie einen Fehler finden, öffnen Sie bitte ein Issue. 

## Pull Requests
Fühlen Sie sich frei, einen Pull Request zu öffnen, um neue Funktionen hinzuzufügen oder Fehler zu beheben. 

## Diskussionen
Wir freuen uns über Rückmeldungen und Vorschläge zu neuen Ideen und Funktionen. Bitte seien Sie geduldig und erwarten Sie, dass wir auf Ihre Anfragen zeitnah reagieren. 

Vielen Dank für Ihre Unterstützung!
